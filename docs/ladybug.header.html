Ladybug Header

class ladybug.header.Header(data_type, unit=None, analysis_period=None, metadata=None)[source]¶

Bases: object

DataCollection header.

Header carries meatdata for DataCollections including data type, unit and analysis period.

Parameters:
  • data_type – A DataType object. (e.g. Temperature)
  • unit – data_type unit (Default: None).
  • analysis_period – A Ladybug analysis period (Defualt: None)
  • metadata – Optional dictionary of additional metadata, containing information such as ‘source’, ‘city’, or ‘zone’.
Properties:
  • data_type
  • unit
  • analysis_period
  • metadata
